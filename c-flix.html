HOW IT WORKS

One BEFLIX artifact, three ways to author it.

MAP — the Voronoi graph. Every territory plays its own BEFLIX reel. Tap a cell to inspect; tap an adjacent cell to cut (the dashed path is your film-of-films). Turn on DRAW to paint the selected reel right on the map; neighbors keep playing.

CODE — the same map decompiled to one BEFLIX-MAP script: each cell is a ;@CELL block of CLR/PNT/LIN/REC, each cut a ;@CUT edge. Edit the text, hit CODE → MAP, and the map rebuilds. This script is also the edit log and the export format — one artifact.

TYPE — paste prose. Each block becomes a territory, cuts wired in reading order, with your chosen motion (typewriter / ripple / fade…) rendered into each reel. Write a screenplay, get a navigable spatial film.

DEEP-EDIT — open the full dot-matrix frame editor on one reel: pen/box/patch, per-reel BEFLIX script, text stamp, ripple, frame ops, onion skin.

RIPPLE works at both scales at once: it propagates across the graph and renders a pixel-ripple into each reel it crosses.
