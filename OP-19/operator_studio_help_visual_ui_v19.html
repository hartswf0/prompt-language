HELP · VISUAL GUIDE + LIVE UI

    
  

  

    

      
D DIRECTUse it to group, title, select, move, and order clips and frames.

      
E PUBLISHUse it to open editable work or write GIF, video, HTML, Twee, PNG, BEFLIX, or EDL files.

    


    
Quick startperform these actions in order

    

      
1
Build. Tap FILM. Add or enter an ACT, SCENE, CLIP, and SHOT. Tap the shot you will change.

      
2
Draw. Press PEN, BOX, or ERASE. Pick ink from BLACK to WHITE. Set SIZE. Drag on the main frame.

      
3
Sequence. Press BLANK FRAME to add the next frame. Press ALTERNATE to fork. Press PRUNE to remove a branch. Press PLAY to run the timeline.

      
4
Direct. Press D. Drag shots or whole clips. BRANCH keeps membership and alternatives. DIRECT can show CURRENT, ALL, or several branch reels. TIMELINE plays the current reel.

      
5
Publish. From DIRECT press PUBLISH, or press E. Choose TIMELINE or DIRECT SELECTION. Then press the file you want written.

    



    
Visual mappress the real interface, or press show me

    
Rule. Each image below is a real crop of the interface. Press SHOW ME to jump to the live control and flash it inside the studio.


    

      

        
1 · Top rail and search

        
        
Use this strip to switch direct views, search the current direct scope, open help, and open the log.

        

          
          
        

      


      

        
2 · Direct viewer

        
        
The large stage shows the current shot. The strip below exposes frames. Direct is for moving, grouping, and ordering.

        

          
          
        

      


      

        
3 · Direct clip field

        
        
These cards are the material you direct. Open all, current, or several branch reels, then reorder shots and segments.

        

          
          
        

      


      

        
4 · Bottom command rail

        
        
The bottom rail selects the operating mode. AUTHOR draws. BRANCH changes structure. DIRECT orders. PUBLISH writes files.

        

          
          
          
        

      

    


    
Control mapeach label names the action it performs

    

      

        
Top rail

        

          
Status
Shows off, ready, connecting, live, or frozen.

          
Role
Shows HOST, GUEST, waiting, or no role.

          
Mic
Turns the microphone track on or off.

          
Camera
Starts local camera and microphone capture.

          
Connect
Creates or joins a shared room.

          
Chat
Opens the text channel.

          
?
Opens this manual.

          
Log
Shows connection and system events.

        

      


      

        
Timeline

        

          
Blank
Adds a blank frame after the current frame.

          
Alternate
Forks a sibling take from the current position.

          
Prune
Deletes the selected branch below the current node.

          
More
Opens node operations: go, fork, add inside, duplicate, prune.

          
Play
Runs the timeline.

          
FPS
Sets playback and export frame rate.

        

      


      

        
Address rail

        

          
FILM
Names the root container.

          
D ACT
Groups scenes.

          
C SCENE
Groups clips.

          
B CLIP
Groups shots into one directed unit.

          
A SHOT
Names one editable frame.

          
Tap
Sets the level used by AUTHOR and BRANCH operations.

        

      


      

        
Bottom commands

        

          
A AUTHOR
Draws, imports, scripts, or generates frames.

          
B BRANCH
Stores which shots belong, which paths fork, and which alternatives remain.

          
C CREW
Connects camera, mic, roles, room, and collaborator.

          
D DIRECT
Orders the shots stored by BRANCH. Every move redraws the timeline immediately.

          
E PUBLISH
Opens projects and writes files.

        

      


      

        
DIRECT

        

          
Current
Shows the branch currently active in BRANCH and TIMELINE.
All
Default DIRECT scope. Shows every shot from every branch so no branch can disappear.
Branch reels
Tap several reels to direct multiple branches together. Press → to make one reel current.
Clips
Groups shots by B CLIP and exposes clip titles and order.

          
Grid
Shows many shot thumbnails.

          
List
Shows file identity and address in rows.

          
Columns
Moves from ACT to SCENE to CLIP to SHOT.

          
Board
Scrolls shots horizontally.

          
Outline
Expands the whole hierarchy.

          
Lock
Prevents a frame from moving.

          
Play
Previews a frame or clip.

          
Publish
Sends the DIRECT selection to E PUBLISH.

        

      


      

        
PUBLISH scopes

        

          
Timeline
Exports the branch reached by PLAY.

          
DIRECT selection
Exports the frames selected in D, in their current order.

          
Whole film
Exports every branch and node.

          
Current frame
Exports only the open shot.

          
Editable
Save project writes a file that OPERATOR can reopen.

          
Playback
GIF, WEBM, and MP4 are viewing files, not editable projects.
