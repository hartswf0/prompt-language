CINEOSIS · SEMIOSIS

An instrument for unlimited semiosis. A seed in language becomes a film, becomes code, becomes a moving image — and the moving image can be read back into language. Meaning never leaves language; it only changes register.

The four registers

SEED — natural language. What you type. The first sign.

FILM — cinematic language. The LLM engine (ABC CINEOSIS) reads the seed through a fixed lexicon: seven image-types (perception, action, affection, opsign, sonsign, crystal, recollection) across three temporal forces (PAST acts on PRESENT acts on FUTURE). The blueprint is the interpretant of the seed.

CODE — formal language. BEFLIX-128: CLR PNT LIN REC SHF on a 128×96 grid. This register is independently meaningful — edit it and RUN to change the image without re-composing. Meaning is use: a command means what it does to the grid.

SCREEN — the visible register. The code executes into dot-matrix motion. It is fully generated from language and re-describable into language — closing the loop to SEED′.

RE-READ — closing the loop

On SCREEN, RE-READ → SEED′ performs the return arrow: the reader turns the finished film back into language and loads it as a new seed. COMPOSE again and the film mutates — Seed → Seed′ → Seed″ — unlimited semiosis you can crank by hand. The LOOP counter tracks how deep you've gone.

Perception toggle (GLYPH / VISION). GLYPH reads the formal string of the dots directly — the claim that the image was always already language. VISION re-sees the rendered plate through a vision pass — the admission that the percept exceeds its code and must be re-seen to be re-spoken. The tension between them is the point.

Fidelity dial. From FAITHFUL (Seed′ closely re-describes this film — echo) to MUTATE (Seed′ keeps only a trace and leaps elsewhere — productive drift). Turn it down and watch meaning travel.

Using it

Write a seed. Pick a mode, or leave FULL SEMIOSIS on to run the whole chain at once. COMPOSE. Read the FILM, inspect or edit the CODE, watch the SCREEN. Change the seed and the film re-composes; change the code and only the image re-runs — proving each register carries meaning on its own terms.

The engine

COMPOSE calls a language model (Claude) acting as the ABC CINEOSIS composer. The AI step runs when this instrument is live inside Claude; the BEFLIX runtime, rendering, and exports are fully local and need no network.
